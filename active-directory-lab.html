Active Directory Enterprise Lab

Windows Server 2022 • User Management • Authentication Troubleshooting

Project Overview

Built and configured a structured Active Directory lab environment simulating a real-world enterprise network. The lab included Windows Server 2022 as a Domain Controller, domain-joined client machines, and department-based organizational units to replicate enterprise identity management workflows.

Environment Configuration

  • Installed and configured Active Directory Domain Services (AD DS)
  • Created domain structure and Organizational Units (OUs)
  • Configured DNS for internal name resolution
  • Joined Windows client machines to the domain

User & Group Management

  • Created and managed user accounts and security groups
  • Assigned group-based access control
  • Implemented password policies and account lockout policies
  • Performed onboarding and offboarding simulations

Incident & Troubleshooting Scenarios

  • Resolved DNS misconfiguration affecting domain connectivity
  • Troubleshot authentication failures and account lockouts
  • Restored network functionality by correcting DNS server settings
  • Validated connectivity using ping and domain resolution tests

Skills Demonstrated

  • Active Directory administration
  • Authentication workflow understanding
  • DNS troubleshooting
  • Network configuration validation
  • Structured documentation of technical findings
